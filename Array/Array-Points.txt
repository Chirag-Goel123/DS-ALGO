3) Difference Array Technique : For Range Problems
    Story : Imagine you have a friend and you asked him to add the number in the array.
            He choose the starting point correct but he stop's only at the end.
            
                      0 1 2 3 4
            Array -> [0,0,0,0,0]
            You asked add 5 from 0 to 2. [0,2,5].
            What He did :
                      0 1 2 3 4
            Array -> [5,5,5,5,5].
            So, you said.Okay. No issues. You make [3,4,-5] now.
                      0 1 2 3 4
            Array -> [5,5,5,-5,-5].

    Intuition : 
            Req -> [1,3,5]. [He chooes the starting point correct]
                    At 1 , he adds 5 and at ending point +1 , he add -1*5. 
                      0 1 2 3 4
            Array -> [0,5,0,0,-5].      

            Ans will be cumulative sum.

    Step 1 : [L,R,X] -> arr[L] -> +X , arr[L+1] -> -X. [ L+1 <arr.length] 
    Step 2 : Cumulative Sum.

    Code :
            for(int[] query : queries)
            {
                int L=query[0];
                int R=query[1];
                int x=query[2];
                arr[L]=;
                if(R+1<n)
                {
                    arr[R+1]=-1*x;
                }
            }
            int sum=arr[0];
            for(int i=1;i<n;i++)
            {
                arr[i]+=sum;
            }
    
    TC : Range Updte : O(1) + Cumulative Sum : O(n).

30) Slow-Fast Technique : (Hair and Tortoise)
    When to use it?
        Jab Duplicacy ki baat ho or jo values hai na vo indexes ki tarah bhi kam kr sakti ho.
    
    Technique : Slow, Fast =0.
    Step 1 -> First detect the cycle.
            Increment slow by 1 and Fast by 2.
            If they meet than, there is for sure cycle is present.
    
    Step 2-> If cycle detected, but we need to find the starting point of the cycle.
            Put slow to the starting point of array/LinkedList. 
            Do not do changes in the fast postion.
            Increment both slow and fast by 1.
            The point where they meet is the starting point.
